Honi buruz

estropadak.eus traineru estropada liga desberdinetako emaitza eta datuak bildu eta era eraginkor batean eskaini nahi dituen proiektu bat da.
Orrialde honetan azaltzen den informazioa hainbat iturri ofizialetatik hartua izan da.

Web gune honetan, traineru estropada liga nagusien informazioa topatu dezakezu:

  • TKE liga (EuskoLabel liga)
  • KAE liga (KAE-1 eta KAE-2 mailak)
  • Euskotren liga (nesken arraunketa)
  • ETE liga (nesken arraunketa)

Kode irekiko proiektua

2020 udazkena ezkeroztik kode irekiko proiektua bihurtu da eta web gunea osatzen duten osagai desberdinen kodea GitHub-en dago eskuragai edonorentzat. Gaurkoz, web gunearen bezeroaren aldeko kodea, web gunearen zerbitzari aldeko kodea eta datuak irakurri eta antolatzeko liburutegia publiko dira.

Proiektuarekin bat egin nahi baduzu, akatsen bat zuzendu nahi baduzu edota zure laguntza eskaini nahi baduzu, ireki "issue" berriak kode biltegietan, edota "issue" hauek konpondu zuen ekarpenaren bidez ("pull request").

Pribatutasuna

Web gune honetan ez dugu cookie-rik erabiltzen, ezta erabiltzaileen jarraipena (tracking) egiten.

Harremana

Gurekin harremanetan jarri nahi baduzu, zure iritzia eman edota iradokizunen bat egin jarri gurekin harremanetan info "a-bildua" estropadak "puntu" eus helbidean edota jarrai gaitzazu Twitter-en @estropadak kontuan.
